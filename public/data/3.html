Ok, I lied, we need to style the table too. See that space in our TH line?

Kick rocks stoopid space!

table {
  border-spacing: 0px;
}

What you just did was override the defaults se by the browser. At some point you will come across the concept in CSS of a reset file.

That is just a file that 'flattens' all the browsers into a single set of settings instead a large set of disperate values. Yeah, they leave it to us supermen (and supergals) to fix these sorts of things.

But now that table is all jammed up against the paragraphs on this page. Let's fix that:

table {
  border-spacing: 0px;
  margin: 2em;
}

Just for 'shits and giggles' as my ol' man used to say, let's make this text bareable again. Typography is very important!

p {
  font: 1.2em helvetica, serif;
  line-height: 1.6;
}
