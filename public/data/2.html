Today, we get an intro to CSS!

See the window pane to the left and on the bottom half? That represents a CSS file and in this case, the one that directly effects the HTML to the left and above. The body section of the CSS will effect everything in the HTML section.

To see how this works, change the text in the CSS window by adding the following text to the: body section.

  padding: 16px;

See how the text moved inward from the container? (The container being everything in the body!) You just styled the HTML using CSS. Now let's make those headings a bit nicer.

Create a new "CSS block" under the body block in the CSS window. It should look like this:

h1 {

}

Then we will change the alignment by changing text-align to center like so:

h1 {
  text-align: center;
}

We can also make things align to the right side of the container. Let's make the h2 tag hang out on the right side of things with the following CSS:

h2 {
  text-align: right;
}

So far we have been styling HTML elements or tags. We can also style those same elements not by their element name, but by an ID or a CLASS. Allow me to demonstrate with the following HTML:

<div id="person">
  <div class="info name">Jim OKelly</div>
  <div class="info email">heyasshole@rubymentor.io</div>
</div>

That was me displaying the code on the page here for you to see. Now comes the real thing that we will style:

Jim OKelly
heyasshole@rubymentor.io

To change those divs, let's start with the outside container that has an ID. The first thing you should know about IDs is that they are unique. If you put two elements on the same page and give them the same ID, don't expect shit to work.

Add this to your CSS window:

div#person {
  margin-left: 1em;
  padding-left: 1em;
  border-left: 1px solid #333;
}

You should now see the whole block of text move to the right, get a vertical line, a bit more padding and then the text. We added both a margin and some padding, and a left border.

You can use margin or margin-left/top/right/bottom to add space outside the container, and padding or padding-left/top/right/bottom to add space inside the container.

You can also use border or border-left/top/right/bottom to add, yep... a border. Notice the arguments to the style, we need a width, a type of line (it could be dotted for instance) and a color.

A color like #333 is what is known as a HEX number color. You can also use colors like red or blue. This applies to all colors in CSS.

Now let's finish up with some changes to a class or two. Add the following CSS:

.info {
  padding: 0.5em;
}

.name {
  font-weight: bold;
}

.email {
  font-style: italic;
}

As you can see, we can effect different elements by adding CSS rules to class names and then applying that class to the element. Classes are not unique and can be added on many different HTML elements.
